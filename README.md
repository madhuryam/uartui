<!-- ratatui -->

Ratatui is a Rust library for building fast, lightweight, and rich terminal user interfaces

<!-- crossterm -->

Crossterm is a pure-rust, terminal manipulation library that makes it possible to write cross-platform text-based interfaces

<!-- tokio/tokio-serial -->

tokio is a runtime for writing reliable asynchronous applications with Rust. It provides async I/O, networking, scheduling, timers, and more

tokio-serial is an asynchronous serial port implementation for Rust's tokio runtime, enabling non-blocking I/O operations on serial devices

<!-- serialport -->

Rust library for serial port communication

<!-- serde -->

Serde is a framework for serializing and deserializing Rust data structures
